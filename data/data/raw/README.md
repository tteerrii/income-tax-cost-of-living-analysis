# Raw Data

This folder contains the original datasets downloaded from external sources.  
These files are **not modified** and serve as the starting point for data cleaning and analysis.

## Datasets

### median-household-income-by-state-2025.csv
Dataset containing median household income by U.S. state.

This dataset provides the income estimates used as the starting point for calculating
net income after state taxes.

---

### 2026 Tax Brackets and Federal Income Tax Rates.csv
State income tax bracket data obtained from the Tax Foundation.

This dataset includes:
- tax bracket ranges
- marginal tax rates
- state-level tax information

The dataset is used to estimate state income tax for different income levels.

---

### grocery-prices-by-state-2026.csv
Dataset containing grocery cost information by U.S. state.

This dataset provides an estimate of average grocery costs used to calculate annual living expenses.

---

### rental.csv
Dataset containing average rental prices by state.

This dataset provides average rent estimates used to calculate annual housing costs in the analysis.
