
# Visualization

This folder contains visual outputs generated from the analysis.

## Files

### tableau_dashboard.png
Screenshot of the Tableau dashboard created for this project.

The dashboard visualizes affordability across U.S. states using the final dataset
`remaining_income_by_state.csv`.

The visualization highlights key variables including:

- average annual net income
- annual rental cost
- annual grocery cost
- remaining income after basic living expenses

These visualizations help compare the cost of living and financial affordability across states.
